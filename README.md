# Share-Experience-For-Github.edu

✅ 申请前确认与准备

在开始申请前，请确保你符合以下条件并准备好相应材料:

1.聪明的大脑，强大的动手能力。

⚠️2.教育邮箱：一个你所在学校分配的、以 .edu 或类似域名结尾的邮箱。使用它注册或关联到你的GitHub账户，能极大提高成功率。

⚠️3.一个注册一段时间的github账号

⚠️4.添加个人账单支付信息(Payment Information) tips:你所填写的内容要确保与之后和学生认证阶段填写的内容完全一致。

⚠️5.添加双因素身份验证,在账户设置的“Password and authentication”中启用(需要下载Authenticator app)

📝 核心申请步骤

1.访问申请入口:https://github.com/settings/education/benefits?locale=en-US

2.填写申请表单，按照页面引导，如实填写你的个人信息。

关键点包括：

在“School name”中，准确填写或选择你的学校。

⚠️在“Email you use for school”中，务必填写你的教育邮箱。

授予页面位置访问权限，这有助于验证你与学校的关联。

3.上传证明材料（这是审核的核心环节）

准备材料:在白纸上用整齐清晰的英文手写(衡水体等等)仿照此张照片![_67_46](https://github.com/user-attachments/assets/be691138-0887-428b-a56d-d170b9d9d100)


选择证明类型：在下拉菜单中选择“Date School ID-Good”

拍摄并上传：使用摄像头清晰拍摄你的证明文件。必须实时拍摄，不能直接上传图片文件。确保文件上的所有关键信息（你的姓名、学校名称、日期等）都清晰可见。

4.提交并等待审核

提交申请后，通常会收到确认邮件。审核时间一般为十分钟以上。

🔄 审核后的操作

1.通过审核：审核通过后，你会在 GitHub账户的“Settings” -> “Billing and licensing” 页面看到“Student verified”的标识。

2.激活权益：72h后权益才会到账。

GitHub Copilot：前往 GitHub Copilot 页面 点击激活即可。

JetBrains 工具：前往 JetBrains 官网的教育页面，用你的GitHub账号登录并关联，即可免费下载使用。

3.审核被拒：如果被拒，邮件通常会说明原因。常见问题包括：

(1)证明材料不清晰、信息不全或非当前在读证明。

(2)个人资料（尤其是姓名）与证明文件不一致。

(3)账户为新注册账号或信息不完整，建议注册满3天以上并完善个人资料（如Bio、学校信息）后再申请。

(4)未开启双重认证（2FA）。在账户设置的“Password and authentication”中启用。
