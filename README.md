# Share-Experience-For-Github.edu

> A practical guide for applying to the GitHub Student Developer Pack.

## 申请前准备

在开始申请前，请先确认你已满足以下条件，并准备好相关材料：

1. **学校邮箱**  
   使用学校分配的教育邮箱（通常以 `.edu` 或学校专属域名结尾）。建议在申请时使用该邮箱关联 GitHub 账户，以提高审核通过率。

2. **已使用一段时间的 GitHub 账号**  
   建议使用注册时间较久、资料较完整的账号进行申请。

3. **补充账户资料**  
   请完善个人资料，包括姓名、头像、学校信息、Bio 等内容，确保与申请材料一致。

4. **添加付款信息**  
   根据 GitHub 要求填写 Payment Information，并确保后续认证阶段提交的信息保持一致。

5. **开启双重认证（2FA）**  
   前往 GitHub 账户设置中的 **Password and authentication** 启用双重认证，并完成 Authenticator App 配置。

## 申请流程

### 1. 进入申请页面

访问 GitHub 教育权益申请入口：

https://github.com/settings/education/benefits?locale=en-US

### 2. 填写申请表单

按照页面提示如实填写信息，重点注意以下内容：
![_67_46](https://github.com/user-attachments/assets/f20a8f9c-b08d-41db-aa50-94fadb95d0c5)

- **School name**：准确填写或选择你的学校名称。
- **Email you use for school**：必须填写你的教育邮箱。
- 如页面要求授权位置访问权限，请根据实际情况授权，以便辅助验证你与学校的关联。

### 3. 上传证明材料

这是审核过程中最关键的一步。请准备清晰、真实、符合要求的证明材料，并通过页面提供的方式完成上传。

建议注意以下事项：

- 证明材料内容应清晰可辨，避免遮挡、模糊或反光。
- 文件中的姓名、学校名称、日期等关键信息应完整可见。
- 上传前确认材料内容与账户资料一致。
- 如页面要求实时拍摄，请按要求使用摄像头完成拍摄，不要直接替换为本地图片。

> 提示：不同学校和不同审核场景下，GitHub 可能接受的证明材料类型会有所不同，请以申请页面的实时提示为准。

### 4. 提交并等待审核

提交后，GitHub 通常会发送确认邮件。审核时间会因申请量和材料完整性而有所不同，请耐心等待。

## 审核通过后的操作

1. **查看认证状态**  
   审核通过后，你可以在 GitHub 账户的 **Settings → Billing and licensing** 页面看到学生认证状态。

2. **激活相关权益**  
   部分权益可能不会立即到账，通常需要等待一段时间后再手动激活或同步。

3. **常见可用权益**  
   例如 GitHub Copilot、JetBrains 教育订阅等，通常可在对应服务页面完成绑定或激活。

## 审核被拒时的常见原因

如果申请未通过，通常会收到邮件说明原因。常见问题包括：

- 证明材料不清晰、信息不完整，或无法证明当前在读状态。
- 账户信息与证明文件不一致，尤其是姓名、学校名称等字段。
- 账户过新或资料不完整，建议完善资料后再重新申请。
- 未开启双重认证（2FA）。

## GitHub Student Developer Pack 的常见权益

以下为 Student Developer Pack 中常见的代表性权益，具体内容可能随官方合作伙伴调整而变化：

### GitHub 核心服务

- **GitHub Pro**：提供高级的协作能力与开发者功能。
- **GitHub Copilot**：AI 编程辅助工具，通常可在权益页面中激活。
- **GitHub Pages**：用于托管静态网站、个人主页和项目文档。

### 开发工具与云服务

- **JetBrains IDE 套件**：可申请 JetBrains 教育许可，用于 IntelliJ IDEA、PyCharm、WebStorm 等产品。
- **Microsoft Azure**：通常可获得一定额度的云服务信用。
- **DigitalOcean**：通常可获得一定额度的云服务信用。
- **AWS Educate**：提供教育用途的云服务资源或学分。
- **Namecheap**：可能包含域名或 SSL 相关优惠权益。

### 学习与教育资源

- **Educative**：编程学习课程权益。
- **DataCamp**：数据科学相关课程权益。
- **Frontend Masters**：前端方向课程权益。
- **GitHub Foundations**：可能包含认证相关的学习或考试支持。

### 其他实用工具

- **Canva Pro**：用于设计演示文稿、海报与视觉内容。
- **1Password**：用于密码与安全信息管理。
- **GitKraken Pro**：增强型 Git 图形化客户端。
